## Regras de curadoria

- Registrar apenas síntese; não versionar export bruto com PII sensível.
- Sempre incluir:
  - janela temporal analisada,
  - volume aproximado de mensagens/fontes,
  - insights acionáveis,
  - riscos/limitações de interpretação.
- Conectar cada insight a possível ação (produto, marketing, operação, automação, conteúdo).

## Fluxo mensal de manutenção (mini playbook)

1. **Ler fontes novas** (chat/docs/comunidade) e registrar janela temporal.
2. **Separar** observação, inferência e recomendação.
3. **Publicar síntese** em `insights-<fonte>-<yyyy-mm>.md`.
4. **Converter** principais achados em oportunidades acionáveis (owner + próximo passo).
5. **Atualizar este README** com novos artefatos para manter navegabilidade.
