# Adventure Work OS

Programa **canônico** da Adventure Labs para operação **híbrida humano + inteligência artificial**: fluxo de trabalho, governança, captura de demandas (estilo GTD) e integração com **Asana**, **`knowledge/`**, **Docs** e ferramentas (Cursor, agentes, MCP, skills).

**Versão do método:** ver `VERSION` e [CHANGELOG.md](./CHANGELOG.md).

**Não é uma aplicação deployável** — é documentação e artefatos versionados no monorepo, alinhada a `apps/core`.
