# Changelog — Adventure Work OS

Versionamento semântico do **método** (não confundir com releases de aplicações).

- **MAJOR** — mudança de fase ou quebra de regra operacional (ex.: novo SSOT).
- **MINOR** — novo módulo fechado ou bloco grande de governança.
- **PATCH** — clarificações, diagramas, checklists.
