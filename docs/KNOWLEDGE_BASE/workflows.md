# Workflows Knowledge Base

## Regra GTD para Dashboard

O ponto operacional de Igor e Mateus no Dashboard deve estar sempre vinculado ao **GID da Task no Asana**.

## Padrao Obrigatorio

Para qualquer item de trabalho rastreado no Dashboard:

1. criar (ou reaproveitar) a task correspondente no Asana;
2. registrar o `task_gid` no item de Dashboard;
3. manter status sincronizado entre Dashboard e Asana;
4. encerrar somente quando a task no Asana estiver concluida.

## Formato Minimo de Vinculo

- `owner`: Igor | Mateus
- `task_gid`: `<asana_task_gid>`
- `project_gid`: `<asana_project_gid>`
- `next_action`: proxima acao GTD clara e verificavel

## Objetivo

Garantir rastreabilidade ponta a ponta (planejamento -> execucao -> fechamento), reduzindo perda de contexto e evitando tarefas sem dono ou sem fechamento formal.
